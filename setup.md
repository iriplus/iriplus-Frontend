# Project Setup Guide
This document explains how to set up the project on your local environment.

## 1. Install Git
If Git is not installed, download it from [Git](https://git-scm.com/downloads)

## 2. Clone the Repository
Choose a directory on your machine and clone the repository by opening a terminal in your chosen folder and typing
    ```git clone (repository_url)```

## 2. Install Python
Make sure you have **Node.JS v20.19.0 or newer** installed.
If Node.JS is not installed, download it from [Node.js](https://nodejs.org/es).

## 3. Install angular dependencies
Once you've installed Node.js and you've cloned the repository, you can install the dependencies with
    ```npm install```

## 5. Run the project
You can now run the frontend project with
    ```npm start```
